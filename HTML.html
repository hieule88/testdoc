HTML
- HTML Hypertext Markup Language tm dch l ngn ng nh du siu vn bn.
Ngi ta thng s dng HTML trong vic phn chia cc on vn, heading, links,
blockquotes,
- HTML document c ui file dng .html hoc htm. Bn c th xem chng bng cc
trnh duyt web hin hnh nh Google Chrome, Firefox, Safari, Nhim v ca trnh
duyt l c nhng file HTML ny v bin i chng thnh mt dng ni dung
visual trn Internet sao cho ngi dng c th xem v hiu c chng.
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>The Main Heading</h1>
<h2>A catchy subheading</h2>
<p>First paragraph</p>
</body>
</html>
Trong :
<!DOCTYPE html>: khai bo kiu d liu hin th
<html> v </html>: cp th bt buc, element cp cao nht, c nhim v ng gi tt
c ni dung ca trang HTML
<head> v </head>: khai bo cc thng tin meta ca trang web nh: tiu trang,
charset
<title> v </title>: cp th nm bn trong th <head>, dng khai bo tiu ca
trang
<body> v </body>: cp th dng ng gi tt c cc ni dung s hin th trn
trang
<h1></h1>, <h2></h2>: nh dng d liu dng heading. Thng thng c 6 cp
heading trong HTML, tri di t <h1> ti <h6>. Trong , <h1> l cp heading
cao nht v <h6> l cp heading thp nht.
<p> v </p>: cp th cha cc on vn bn ca trang web
